# Sunday Order

The schedule for each Sunday class session.

---

## Timeline

| Time | Activity |
|------|----------|
| 9:15 | First worship service ends |
| 9:25+ | Arrive at classroom, get food, fellowship |
| 9:45 | **Class Begins** — Opening: welcome, announcements, food volunteers, prayer requests, prayer |
| ~9:55 | **Teaching** — Lesson begins |
| 10:45 | **Class Ends** |
| 10:45 - 10:55 | Store belongings, cleanup |
| 10:55 - 11:05 | Walk to worship center, find seats |
| 11:05 | Second worship service begins |

---

## Opening (9:45 - ~9:55)

Led by the class leader:
1. Welcome everyone
2. Announcements
3. Ask for food/refreshment volunteers for next Sunday
4. Prayer requests and updates
5. Opening prayer

---

## Teaching Time (~9:55 - 10:45)

- ~50 minutes for the lesson
- Led by the assigned teacher for that week
- See [Guidelines](Guidelines.md) for teaching approach

---

## Dismissal (10:45)

- End on time so people can:
  - Store belongings
  - Help with cleanup
  - Walk to the worship center
  - Find seats before 11:05 service
